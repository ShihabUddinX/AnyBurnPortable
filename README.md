Short Description:
AnyBurn Portable is a lightweight, free CD/DVD/Blu-ray burning software for Windows.  
It requires no installation and runs directly from a USB drive.

Description:
AnyBurn Portable allows you to:
- Burn data to CD/DVD/Blu-ray discs
- Create Audio CDs and MP3 discs
- Rip Audio CDs to MP3, FLAC, WAV, etc.
- Create and convert disc image files (ISO, BIN, CUE)
- Copy discs directly from one to another
- Create bootable USB drives and install Windows

Usage Notes:
- Portable version does not require admin rights.  
- Works on Windows XP, Vista, 7, 8, 10, 11 (32-bit & 64-bit).  
- For help or updates, visit the official website or PortableApps.com forum.
